Connector Object Java Examples

The following examples introduce the capabilities of the Java connector object:

  • Hello World Example with a Java Connector Object
  • Data Manipulation with a Java Connector Object
  • IDL Command Line with Java Connector Object

Note: The Java class files needed for the Export Bridge are found in the com.idl.javaidl package, which is stored in the javaidlb.jar file. See "Preparing to Use the IDL Connector Object” for more information.

Note on Running the Java Examples

Examples in this chapter provide Windows-style compile javac (compile) and java (run) commands. If you are running on a platform other than Windows, use your platform’s path and directory separators and see Java Requirements for information about the bridge_setup file, which sets additional information.
