iTools Help

The iTools make up the IDL legacy visualization interface. These tools have largely been replaced by IDL Graphics, but are still available to use.

For more information, see the iTools Routines.

You can learn more about iTools from these PDF resources, found in the help\pdf directory of your IDL installation:

  • iTools User Guide - how to use the iTools routines to visualize data
  • iTools Developer Guide - how to develop iTools applications
