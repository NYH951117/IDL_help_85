Animation

CW_ANIMATE - Creates a compound widget for animation. (Direct Graphics)

CW_ANIMATE_GETP - Gets pixmap window IDs used by CW_ANIMATE. (Direct Graphics)

CW_ANIMATE_LOAD - Loads images into CW_ANIMATE. (Direct Graphics)

CW_ANIMATE_RUN - Displays images loaded into CW_ANIMATE. (Direct Graphics)

IDLffMJPEG2000 - Provides the ability to read and write Motion JPEG2000 (MJ2) files.

IDLffVideoWrite - Creates a video file with an optional soundtrack.

FLICK - Causes the display to flicker between two images. (Direct Graphics)

XINTERANIMATE - Displays animated sequence of images.
