Routines (by topic) > Dialog Routines

Dialog Routines

DIALOG_COLORPICKER - Creates a dialog from which to select colors.

DIALOG_MESSAGE - Creates a modal message dialog.

DIALOG_PICKFILE - Creates a native file-selection dialog.

DIALOG_PRINTERSETUP - Opens a native printer property dialog.

DIALOG_PRINTJOB - Opens a native printer dialog.

DIALOG_READ_IMAGE - Open a dialog for reading image files.

DIALOG_WRITE_IMAGE - Open a dialog for writing image files.
