IDLnetURL::GetProperty

The IDLnetURL::GetProperty procedure method retrieves the value of a property for the IDLnetURL object.

Syntax

Obj->[IDLnetURL::]GetProperty [, PROPERTY = value]

Return Value

None.

Arguments

None.

Keywords

Any property listed under IDLnetURL Properties that contains the word “Yes” in the “Get” row of the properties table can be retrieved using this method. To retrieve the value of a property, specify the property name as a keyword set equal to a named variable that will contain the value of the property.
